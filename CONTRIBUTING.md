# Contributing

Thank you for your interest in contributing to Good Morning AI.

## How to Contribute

1. Fork the repository.
2. Create a new branch.
3. Make your changes.
4. Test your changes.
5. Submit a pull request.

## Areas for Contribution

* Hindi text rendering
* Additional style templates
* Image enhancement
* New CrewAI agents
* UI improvements
* Documentation

## Coding Guidelines

* Follow PEP 8
* Add comments where appropriate
* Test before submitting
