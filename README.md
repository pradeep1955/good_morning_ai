# Good Morning AI

A CrewAI-powered application that:

- Selects an image
- Selects a quote
- Selects a style
- Generates a greeting message
- Generates style instructions
- Renders a final greeting card image

## Technologies

- Python
- CrewAI
- OpenRouter
- Pillow

## Setup

1. Create virtual environment
2. Install requirements
3. Create `.env`
4. Run:

```bash
python crew.py
